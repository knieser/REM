# Robust Expectation-Maximization (REM)

## FA
This folder contains code for running simulations
comparing EM and REM estimation of factor structures
within heterogeneous populations.

Includes:
- FA_sim_main
- FA_data_sim
- SigmaSim
- SigmaSim_Normal
- computeCongruence
- FA_sample_data
- FA_estimates
- EMAlg
- RobustEMAlg
- checkEpsFA
- aggregate
- PrettyFig
